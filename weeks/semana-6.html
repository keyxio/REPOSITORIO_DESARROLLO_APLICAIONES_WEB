Reflexión

¿Qué aprendí? Aprendí que existen diversas maneras de estilizar componentes en React. Se puede optar por CSS puro (Hojas de Estilo, Módulos CSS y Estilos Inline) o utilizar bibliotecas CSS como Tailwind CSS. Tailwind CSS ofrece un enfoque utility-first que permite crear diseños personalizados rápidamente sin escribir CSS personalizado. ¿Cómo aprendí? A través de la lectura y análisis de las fuentes proporcionadas. La información estaba organizada de manera clara, explicando cada método de estilo con definiciones, ejemplos de código y pasos a seguir.
