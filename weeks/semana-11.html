Reflexión

¿Qué aprendí? Django facilita la construcción ordenada de aplicaciones web, separando responsabilidades y agilizando el desarrollo. Las vistas y plantillas permiten adaptar la interfaz según el contexto. El patrón MVT (Model-View-Template) proporciona una estructura clara y mantenible. ¿Cómo aprendí? Armando el esqueleto de un proyecto Django y comprendiendo cómo se conectan las partes desde la URL hasta el HTML final, experimentando con diferentes tipos de vistas y plantillas.
