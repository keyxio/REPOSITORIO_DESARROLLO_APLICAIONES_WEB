Reflexión

¿Qué aprendí? Aprendí a crear interfaces para la comunicación entre sistemas de manera estandarizada. La serialización convierte datos en formatos accesibles para aplicaciones externas. Los principios REST proporcionan una arquitectura clara y escalable para servicios web. ¿Cómo aprendí? Construyendo endpoints y conectándolos con clientes que consumen información desde distintos entornos, probando la interoperabilidad entre diferentes tecnologías.
