Reflexión

¿Qué aprendí? Entendí cómo se procesan los datos en el servidor y cómo se construyen respuestas dinámicas. La lógica backend complementa el frontend, permitiendo interacción real con sistemas y datos. PHP ofrece una sintaxis accesible para principiantes, mientras que JSP proporciona la robustez del ecosistema Java. ¿Cómo aprendí? Programando scripts de backend y analizando la diferencia entre tecnologías, mientras conectaba formularios con bases de datos reales para ver el flujo completo de información.
