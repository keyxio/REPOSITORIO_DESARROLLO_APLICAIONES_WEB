Reflexión

¿Qué aprendí? La capacidad de crear componentes reutilizables es fundamental en React. Esto permite dividir la interfaz de usuario en piezas independientes y reutilizables, lo que mejora la organización del código y la eficiencia del desarrollo. Existen herramientas que simplifican la configuración del entorno de desarrollo de React. Algunas de las más populares son Create React App, Vite y Next.js. Estas herramientas facilitan la creación de un proyecto, la configuración de Webpack, Babel y otras dependencias necesarias. La importancia de herramientas como Babel, que transpila código JavaScript moderno a versiones compatibles con navegadores antiguos, ESLint, que analiza el código en busca de errores y problemas de estilo, y Webpack, que empaqueta y optimiza los archivos de la aplicación para su despliegue. ¿Cómo aprendí? He aprendido sobre React a través del análisis del documento de la clase, este documento ofrece una visión general de React, su historia, sus características principales, las herramientas utilizadas para el desarrollo con React y las diferentes formas de crear un proyecto. Además de replicar el ejemplo en Visual Studio Code.
