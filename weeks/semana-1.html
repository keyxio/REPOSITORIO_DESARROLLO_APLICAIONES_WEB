Reflexión

¿Qué aprendí? Aprendí sobre las tecnologías esenciales para el desarrollo web, incluyendo HTML para la estructura del contenido, CSS para la presentación visual, JavaScript para la interactividad y Web APIs para acceder a funcionalidades avanzadas del navegador y del dispositivo. ¿Cómo aprendí? Las prácticas de laboratorio para Visual Studio Code, me permitieron aplicar los conocimientos teóricos en un entorno práctico, experimentando con las herramientas y tecnologías.
