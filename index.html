Reflexión Metacognitiva

¿Qué aprendí durante este curso?

Durante este curso de Desarrollo de Aplicaciones Web, he experimentado un crecimiento significativo en mis conocimientos sobre tecnologías web fundamentales. Desde la primera semana, donde aprendí los conceptos básicos de las tecnologías web y los diferentes roles en el desarrollo, hasta las últimas semanas donde dominé frameworks como Bootstrap y Tailwind CSS.

Más allá de las tecnologías específicas como HTML5, CSS3, y los frameworks, desarrollé una comprensión profunda sobre la estructura del DOM, la importancia del diseño responsivo, y cómo crear interfaces de usuario modernas y accesibles. También aprendí sobre las mejores prácticas en el desarrollo web y la importancia de escribir código limpio y mantenible.

¿Cómo aplicaré estos conocimientos?

Los conocimientos adquiridos en este curso serán fundamentales para mi desarrollo profesional como Ingeniera de Sistemas. Planeo aplicar estas habilidades en proyectos reales, tanto académicos como profesionales, creando aplicaciones web que resuelvan problemas específicos en mi área de interés: tecnologías de la información y ciberseguridad.

También utilizaré estas habilidades para crear herramientas web que apoyen mis otros intereses como el hacking ético y la continuidad de TI. Además, estos conocimientos me servirán como base para continuar aprendiendo tecnologías más avanzadas como JavaScript, React y desarrollo backend.

Este curso ha sido fundamental para complementar mi formación en Ingeniería de Sistemas. Me ha permitido entender cómo las tecnologías web se integran con otros aspectos de la informática que me interesan, como la seguridad y las redes.
